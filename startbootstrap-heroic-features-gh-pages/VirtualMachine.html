Máquina Virtual

Las máquinas virtuales nos permiten la visualización de una máquina con Oracle VM VirtualBox o con VMware Workstation. Estos son softwares que nos permiten tener máquinas virtuales. De este modo podremos conservar nuestra maquina principal con su Sistema operativo (SO) base y a su vez nos permite tener otro un Sistema operativo dentro del acuel.

Proceso de instalación

Para caso practico en esta página se descargará el software de VMware para ello ingresaremos a su link para descargar descargar aquí

Despues

Una vez que ya se haya instalado el programa, prodeceremos a crear una maquina virtual. Para esto daremos clic sobre la lista desplegable llamada "player".

Aqui nos pediran asignar el espacio en memoria límite para la maquina.

En el ejemplo se colocó un máximo de 60gb. Tabien marcamos la opción que no parte el disco en dintas partes.
