Procreso de instalación de Parrot

Una vez que ya tengamos a nuestra maquina virtual con el ISO instalado, toca ejecutar el sistema operativo para continuar con las instalción interna. En cuanto se ejecute parrot, veremos que la interfaz es muy similar a la de windows, pero primero de seguir explorando el SO, debemos de dar doble clic sobre la carpeta que se llama "instalar". De ahi se nos abrirá la ventana que se muerta en la imagen 1

Usuario

Luego de haber terminado en la pertaña anterior nos pedirá que escojamos un nombre de usuario para iniciar el SO entre otros campos incluyendo la contraseña. En esta pestaña desmarcaremos la opción de iniciar sin contraseña. Y finalizamos esta parte dándole en siguiente en la parte inferior derecha.

Resumen

En cuanto terminamos de hacer el paso anterior podremos ver un resumen de lo que hemos colocado para nuestra distro de Linux. Aquí solo queda darle en siguiente si estamos de acuerdo con la configuración que nosotros mismos hemos elegido.

Instalación

Una vez de corroborar nuestra configuración en la pestaña de Resumen, comenzará a instalarse parrot con nuestras preferencias y lo cual tomará unos cuantos minutos, así que, tocará esperar a que terminé de instalarse este grandioso Sistema Operativo.

Una vez que se haya terminado de instalar nos pedirá reiniciar el equipo, la máquina virtual, y después de eso ya podremos comenzar a utilizar este sistema operativo.
