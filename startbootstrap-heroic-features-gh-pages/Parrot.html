¿Qué es Parrot?

Parrot OS es una distribución de Linux que se enfoca en la ciber seguridad. Está diseñado para pruebas de penetración, evaluación y análisis de vulnerabilidades, análisis forense de computadoras, navegación web anónima, y practicar criptografía.

Creado en 2013

Parrot Security OS fue creado por el equipo Frozenbox. La primera versión fue lanzada el 10 de abril de 2013.Parrot Security OS está disponible en una variedad de ediciones, incluyendo una edición principal, una edición para uso general y una edición para dispositivos móviles.

Ventajas

  • Tiene una amplia gama de herramientas de seguridad.
  • Es una edición portátil.
  • Es encuentra disponible para dispositivos móviles.
  • Tinen gran variedad de edicones.
